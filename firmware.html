🔧 SNIN ESP32 — Firmware v1.0

Прошивка для прямого подключения к Nostr relay · kind:8010 · kind:8012

📋 Что делает прошивка

  • ✅ Подключается к WiFi
  • ✅ Читает DHT22 (температура + влажность)
  • ✅ Публикует kind:8010 в relay-snin.v2.site
  • ✅ Принимает команды kind:8012 от бота @Snindaobot
  • ✅ Отправляет алерты kind:8011
  • ✅ Регистрируется kind:8014 при старте
  • ✅ Мониторинг батареи через ADC

🔌 Подключение

ESP32ДатчикЦвет (типовой)
3.3VDHT22 VCC🔴 Красный
GNDDHT22 GND⚫ Чёрный
GPIO4DHT22 DATA🟡 Жёлтый
GPIO35Батарея ADC🟠 (через делитель)
💡 DHT22 можно заменить на BME280 (изменить тип в коде).
🔋 Для батареи нужен делитель напряжения (2x 100kΩ).

⚡ Веб-прошивка (ESP Web Tools)

Подключите ESP32 через USB к компьютеру, нажмите кнопку —
прошивка установится через браузер (Chrome/Edge).

После прошивки:
  1. ESP32 перезагрузится и создаст WiFi SNIN_Setup
  2. Подключитесь к нему (пароль: snin1234)
  3. Откройте 192.168.4.1 — введите ваш WiFi + relay
  4. ESP32 подключится к relay и начнёт публиковать данные
  5. В боте @Snindaobot: /watchdev esp32_sensor_01
